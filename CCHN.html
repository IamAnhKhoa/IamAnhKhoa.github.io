Xuất Dữ Liệu Cổng BHXH

Nghiệp vụ & Cơ chế lấy dữ liệu:

  • Hệ thống sẽ xuất ra 2 file chuẩn XML/Excel theo định dạng Cổng Giám Định.
  • File 1 (IMPORT_PKDK): Dùng để import danh sách nhân sự ban đầu.
  • File 2 (NVUP_PKDK): Dùng để cập nhật bổ sung nhân sự.
  • Lưu ý quan trọng: Với nhân sự có nhiều dòng lịch sử công tác (xuống dòng), hệ thống sẽ tự động lấy dòng cuối cùng (thông tin mới nhất) để đưa vào file xuất.
